# Dokumentaatio-repo
Dokumentaation repositorio. 

Projektin tavoitteena on uudistaa TutorPalvelu TuPa Oy:n koko järjestelmä nykyaikaiseksi verkkopalveluksi, joka mahdollistaa tutorien ja asiakkaiden välisen vuorovaikutuksen, ajanvaraukset, maksut ja raportoinnin sujuvasti samassa ympäristössä.
Palvelun tekninen toiminta koostuisi kahdesta osasta: Spring boot back end ja React front end. Nämä olisivat toisiinsa yhteydessä pilvipalvelun avulla. Front end olisi se, mitä asiakas näkisi ja back end hoitaisi kaiken logiikan.
