# Dokumentaatio
Projektin nimi on TuPa
Projektin tavoitteena on luoda kuvitteelliselle TutorPalvelu TuPa Oy:lle järjestelmä, joka mahdollistaa tutorien ja asiakkaiden välisen vuorovaikutuksen, ajanvaraukset, maksut ja raportoinnin sujuvasti samassa ympäristössä.
Palvelun tekninen toiminta koostuisi kahdesta osasta: Spring boot back end ja React front end. Nämä olisivat toisiinsa yhteydessä pilvipalvelun avulla. Front end olisi se, mitä asiakas näkisi ja back end hoitaisi kaiken logiikan.
Toiminnallisuuksia joita tulee olla:
- Rekisteröityminen ja sisäänkirjautuminen
- Ajanvaraus tuutorilta
- Nähdä vapaat ja varatut tuutorit
- Tuutori voi laskuttaa asiakasta
- Käyttäjä voi katsella ja muokata omia tietojaan
- Tuutori voi hallita saatavuuttaan
- Ylläpitäjä voi poistaa, muokata tai päivittää muita käyttäjiä
- Ylläpitävä voi tarkastella ja hyväksyä tai hylätä tutoreita
  
Toteutusteknologioita:
- Spring boot
- React
- Jokin pilvipalvelu (Rahti, Render yms.)
